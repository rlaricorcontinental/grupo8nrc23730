Creación de Fluent UI Dark CSS para Markdown

Fluent UI es el lenguaje de diseño moderno de Microsoft, caracterizado por su estética limpia, tipografía clara y efectos sencillos.

En este proyecto, se diseñó un tema oscuro inspirado en Fluent UI, adaptado para ser utilizado en sitios generados con Jekyll y Markdown, sin necesidad de librerías externas ni JavaScript adicional.

1. Objetivo del Estilo

  • Respetar la tipografía "Segoe UI".
  • Mantener un modo oscuro accesible.
  • Incluir botones, tarjetas y enlaces con efectos suaves.
  • Mejorar estilos de código y tablas.
  • Evitar pérdida de rendimiento en la carga del portafolio.

2. Estructura del Archivo CSS

1. Estilos Base

body {
  font-family: "Segoe UI", Tahoma, Geneva, Verdana, sans-serif;
  background-color: #1e1e1e;
  color: #f3f2f1;
  margin: 0;
  padding: 0;
  line-height: 1.6;
}

2. Barra de Navegación

nav {
  background-color: #005a9e;
  color: #ffffff;
  padding: 16px 20px;
  box-shadow: 0 2px 6px rgba(0, 0, 0, 0.3);
}

3. Botones

button, .btn {
  background-color: #0078d4;
  color: #ffffff;
  font-weight: 600;
  padding: 10px 16px;
  border-radius: 6px;
  border: none;
  cursor: pointer;
  transition: background 0.3s ease, transform 0.1s ease;
}
button:hover {
  background-color: #005a9e;
  transform: scale(1.03);
}

4. Tablas

table {
  width: 100%;
  border-collapse: collapse;
  background-color: #252526;
}
th, td {
  padding: 12px;
  border: 1px solid #3e3e3e;
}
th {
  background-color: #0078d4;
  color: #ffffff;
}

3. Implementación

Para aplicar este CSS a todo el sitio:

  1. Guardar el archivo como assets/css/fluentui-dark.css.
  2. Incluir el siguiente <link> en el <head> de tu HTML: 
    <link rel="stylesheet" href="/assets/css/fluentui-dark.css">

Resultado Final

Al aplicar lo anterior, este portafolio obtiene un estilo similar a Fluent UI siendo desarrollado en MarkDown.

Conclusión

Este estilo brinda una experiencia visual limpia y moderna. Al no utilizar librerías externas ni JS adicional, permite una navegación fluida y ligera.
